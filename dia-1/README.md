#  O que é Lógica de Programação?

Lógica de programação é a forma de organizar pensamentos e passos para resolver problemas usando programação.

Programar não é apenas escrever código.

Programar é:
1. Pensar
2. Analisar
3. Resolver problemas
4. Criar soluções

---

#  Exemplo do Dia a Dia

Imagine fazer chá 

Passos:
1. Ferver água
2. Colocar açúcar
3. Colocar chávena
4. Adicionar água quente

Isso é um algoritmo:
uma sequência de passos para atingir um objetivo.

---

#  O que é Algoritmo?

Algoritmo é:
> Uma sequência lógica de passos para resolver um problema.

Todo programa é um algoritmo.

---

#  Conhecendo o Visualg

Visualg é uma ferramenta usada para aprender lógica de programação utilizando pseudocódigo.

Com ele podemos:
- Criar algoritmos
- Testar lógica
- Aprender programação mais facilmente

Download:
https://visualg3.com.br/baixar-o-visualg3-0/

#  Estrutura Básica de um Algoritmo

```visualg
algoritmo "nome"

inicio

// comandos

fimalgoritmo
```

---

#  Primeiro Programa

```visualg
algoritmo "ola_mundo"

inicio

escreval("Olá Mundo!")

fimalgoritmo
```

---

#  Explicação do Código

## `algoritmo`
Define o nome do programa.

## `inicio`
Marca o início da execução.

## `escreval`
Mostra mensagens na tela.

## `fimalgoritmo`
Finaliza o programa.
