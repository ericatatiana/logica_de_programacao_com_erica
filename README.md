#  Exemplo de Código

```visualg
algoritmo "media"

var
n1, n2, media: real

inicio

escreva("Digite a primeira nota: ")
leia(n1)

escreva("Digite a segunda nota: ")
leia(n2)

media <- (n1 + n2) / 2

escreval("Média final: ", media)

fimalgoritmo
```
